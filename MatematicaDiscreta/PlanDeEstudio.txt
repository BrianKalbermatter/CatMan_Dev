📌 1. PROPOSICIONES LÓGICAS
📖 Teoría Resumida
Proposición: enunciado que puede ser verdadero (V) o falso (F).

Conectores:

¬p (negación),

p ∧ q (conjunción),

p ∨ q (disyunción),

p → q (condicional),

p ↔ q (bicondicional).

Tabla de verdad: se usa para analizar la validez.

📌 2. ÁLGEBRA BOOLEANA
📖 Teoría Resumida

Operaciones:

A + B = A OR B

A ⋅ B = A AND B

A’ = NOT A

Leyes:

Idempotencia: A + A = A

Dominación: A ⋅ 0 = 0

Identidad: A + 0 = A

Involución: (A’)’ = A

📌 3. SISTEMAS DE NUMERACIÓN
📖 Teoría Resumida
Bases comunes:

Binario (base 2), Octal (base 8), Decimal (base 10), Hexadecimal (base 16)

Conversión:

Decimal → Binario: dividir por 2 hasta 0, anotar restos.

Binario → Decimal: multiplicar cada bit por su peso (potencia de 2).

📝 Ejercicios
a) Decimal → Binario:
45₁₀ = ?
→ 101101₂ ✅

b) Binario → Decimal:
1101₂ = ?
= 1×8 + 1×4 + 0×2 + 1×1 = 13 ✅

c) Sumar binarios:
1101 + 0110
→ Resultado: 10011₂ ✅
