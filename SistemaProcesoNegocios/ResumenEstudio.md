# Que se conoce como teoria de Sistemas?
- Se conoce como Teoria de sistemas o Teoria General de Sistemas (T.G.S.) al estudio de los sistemas en general, desde una perspectiva interdisiplinaria, o sea, que abarca distintas diciplinas.
Se puede definir un sistema como un conjunto de elemento dinamicamente relacionados entre si, realizando una actividad para alcanzar un objetivo, operando sobre entradas (informacion, energia o materia) y proveyendo salidas (informacion, energia o materia) procesadas, al medio ambiente en el cual esta inserto. Los elementos son el organo de que lo componen. Formando actividades que es la operacion o procesamiento del sistema para alcanzar un objetivo, o sea la finalidad del sistema, operando sobre datos, energia y materia. Para proveer informacion, energia o materia que son las salidas del sistema.

El sistema esta inserto en un determinado ambiente o medio(entorno) con el cual intercambia o no materia, energia o informacion de acuerdo a su condicion de abierto o cerrado.
En resumen el sistema es un: 
        Conjunto de elementos.
        Relacionados Dinamicamente.
        Formando una actividad.
        Para alcanzar un objetivo.
        Operando sobre datos, energia y materia 
        Para proveer informacion, energia y materia.

- Se destaca en identificar los diversos elementos y tendencias indetificables y reconocibles de los sistemas.
Puede definirse como una forma ordenada y cientifica de aproximacion y representacion del mundo real, y simultaneamente, como una orientacion hacia una practica estimulante para formas de trabajo transdisciplinado. Se distingue por su perspectiva  integradora, donde se considera importante la interaccion y los conjuntos que a partir de ella brotan. Gracias a la practica, la TGS crea un ambiente ideal para la socializacion e intercambio de informacion entre especialistas y especialidades.

# Todo Sistema se compone de:
- **Entradas, insumos o inputs.** Son aquellos procesos que incorporan informacion, energia o materia al sistema, proviene de afuera.
- **Salidas, productos u outputs.** Son obtenidos mediante el funcionamiento del sistema y que por lo general salen del sistema al medio externo.
- **Transformadores, procesadores o thoughput.** Mecanismos del sistema que producen cambios o convierten entradas en salidas.
- **Retroalimentacion.** Aquellos casos en que el sistema convierte sus salidas en entradas.
- **Medio ambiente.** Todo lo que rodea al sistema y existe fuera de el, lo cual a su vez constituye un sistema dentro de otro sistema y asi hasta el infinito.

## Se fundamenta en tres **PREMISAS BASICAS**:
# SACADO DEL APUNTE
- **Los sistemas existen dentro de sistemas :** 
        Las moleculas existen dentro de **celulas**, las celulas dentro de **tejidos**, los tejidos dentro de **organos**, los organos dentro de **organismos**, los organismos dentro de **colonias**, las colonias dentro de **culturas nutrientes**, las culturas dentro de **conjuntos mayores de culturas** y asi **sucesivamente**.
- **Los sistemas son abiertos :**
        Cada sistema que se examine, excepto el menor o mayor, recibe y descarga algo en los otros sistemas, generalmente en aquellos que le son contiguos. Los sistemas abiertos son caracterizados por un proceso de intercambio infinito con su ambiente, que son los otros sistemas. Cuando el intercambio cesa, el sistema se desintegra, esto es, pierde sus fuentes de energia.
- **Las funciones de un sistema depende de su estructura :**
        Para los sistemas biologicos y mecanicos esta afirmacion es intuitiva. Los tejidos musculares, por ejemplo, se contraen porque estan constituidos por una estructura celular que permite contracciones.

# SACADO DE INTERNET
## **Tres premisas basicas de los sistemas:**
- 1. **Todo Sistema tiene un proposito u objetivo**
        Todo sistema existe para cumplir una funcion dentro de un entorno mas amplio. Ejemplo: un sistema contable tiene el proposito de registrar y controlar las operaciones financieras.
- 2. **Todo Sistema tiene partes interrelacionadas (Subsistemas)**
        Un sistema no es una unidad aislada, sino que esta compuesto por elementos o procesos que trabajan juntos. Si una parte falla, puede afecta a todo el sistema.
- 3. **Todo sistema opera dentro de un entorno**
        Ningun sistema funciona completamente aislado. Interactua con su entorno (otros sistemas, clientes, gobierno, mercado). Esto da luga a entradas (inputs) y salidas (outputs).

2. **Medio y los limites de un sistema**: El medio se refiere al conjunto de objetos exteriores que rodean, contienen o influyen al sistema. El medio estaria integrado por todos los sistemas sobre los cuales no se tienen control y que no estan integrado por todos los sistemas. Los sistemas abiertos interactuan con otros sistemas.
- La definicion de limites de sistema determina cuales sistemas se consideran bajo control. Entre el sistema y el medio no hay lineas claras de separacion.
- A medida que ocurren, la propia influencia del sistema tiene una retroalimentacion (feedback).
Sucede un proceso constante y de adaptacion, evolucion del sistema que responde tanto a lo interno como a lo externo 

3. **Elementos**: Son las unidades u objetos que forman el sistema.

4. **Relaciones entre unidades**: Son los lazos que unen los elementos (objetos) entre si. Alguna forma de interaccion o interdependecia.

5. **ENTRADAS (INPUTS)**: Son los elementos que ingresan al sistema desde el medio. Es la fuerza de arranque o de partida del sistema que provee el matrial o energia para la operacion del sistema. La entrada puede constituirse de uno o mas de estos: informacion, energia y materiales.

## Informacion: 
- Reduce la incertidumbre: Un sistema necesita saber que esta pasando para poder reaccionar correctamente. Cuando no hay informacion, **el sistema no sabe que decision tomar o como responder**.
Ejemplo: Un semaforo inteligente(Sistema de Trafico):
        - Un semaforo inteligente usa camaras o sensores para detextar la cantidad de autos que hay en cada carril. Si hay muchos en una calle, el sistema alarga el tiempo en verde para reducir el trafico. Si no hay autos, cambia mas rapido al rojo para dar paso a otras calles.
        - **Que pasa si no tiene informacion (FALLAN LOS SENSORES)?**
        El semaforo no puede decidir correctamente cuanto tiempo deja cada luz, lo que puede causar embotellamiento o incluso accidentes.

## Energia:
- Se utiliza para mover y dinamizar el sistema, haciendolo funcionar.
Ejemplo: El cuerpo humano como sistema:
        - El cuerpo humano es un sistema biologico. La energia que necesita para funcionar proviene de los alimentos que consumimos. Esos alimentos se transforman en glucosa, que luego se convierte en energia (ATP) mediante procesos como la respiracion celular.
        - Sin energia el cuerpo no puede moverse por lo tanto se apagaria como una maquina sin combustible.

## Materiales: 
- Son los recursos a ser utilizados por el sistema, como medio para producir las salidas (producto o servicios). Los materiales son llamados operacionales cuando utilizados para transofrmar o convertir otros recursos.
Ejemplo:
        - Maquinas, equipos, instalaciones, herramientas, instrucciones, utensillos y son llamados productivos o materia prima, cuando se transforman o convierten en salidas (productos o servicios).

6. **Proceso de Transformacion**: (thoughput) es el fenomeno que podruce cambios, es el mecanismo de conversion de las entradas en salidas o resultados. Los elementos de entrada se convierten en elementos de salida, agregando valor y utilidad a aquellos. Este proceso de transformacion es el contenido del sistema total y de cada uno de los subsistemasl. Hay entre los subsistemas y los elementos, procesos de accion, de interaccion y de reaccion, siendo estos muy dinamicos.

7. **Salidas**: (outputs) Son los resultados del porceso de transformacion y pueden ser resultados o beneficios, volcado en el medio, influyen sobre el y generaran nuevas entradas al sistema. Las salidas deben ser congruentes con el objetivo del sistema. Los resultados de los sistemas son finales, mientras que los resultados de los **subsistemas** son intermedios.

8. **Proceso de Retroalimentacion**: (feedback) Es la funcion del sistema que tiende a comparar la salida con un criterio o un estandar previamente establecido. **Tiene como objetivo el control** (monitorear) guia y direccion. Registro de Calidad. Es tambien llamado Servomecanismo.
        -Las principales funciones de la retroalimentacion son:
                a). Controlar la salida para enviar mensajes al regulador de entradas.
                b). Cuando algo del exterior afecta al sistema, este detecta el cambio y reacciona para corregirse y mantener su equilibrio o funcionamiento normal. La retroalimentacion en esta funcion lo que hace es detectar cambios, los analiza y corrige el sistema para que no se descontrole. Es como un autocontrol para que el sistema no se desestabilice por lo que pasa afuera.
Como la retroalimentacion es basicamente una accion que el efecto de salida repercute en la entrada, podemos identificar dos tipos: retroalimentacion positiva y retroalimentacion negativas:
---
- **Retroalimentacion Positiva**: Es cuando una salida de un sistema provoca un cambio que refuerza o aumenta aun mas esa misma salida.
- Ejemplo: **Imagina que estas vendiendo jugos:**
- Empiezas vendiendo 10 por dia.
- Los clientes los recomiendan -> mas gente lo compra.
- Vendes 20 por dia -> te emocionas y haces mas publicidad.
- Vendes 50 por dia -> contratas a mas gente y producis mas.
Esa cadena de mas ventas -> mas produccion -> mas venta es una **retroalimentacion positiva**. 
---
- **Retroalimentacion Negativa**: Es cuando el sistema detecta un cambio y actua para corregirlo, disminuyendo o frenando lo que esta pasando, para mantener el equilibrio. Es un control automatico.
- Ejemplo: **En palabras faciles:**
Si algo sube mucho, el sistema actua para que baje.
Si algo baja mucho, el sistema actua para que suba.
Corrige los excesos o faltas.
En el caso de ventas las ventas bajan. Se quedan con mucho productos en stock. El sistema reduce la produccion para no llenar mas el almacen. Con el tiempo, el inventario se ajusta al nivel real de ventas.
Eso es una retroalimentacion negativa:
Se frena una accion, produccion para corregir un problema que es el exceso de inventario.

# Suprasistema
Es necesario definir muy bien las fronteras del sistema y de sus subsistemas. Es el entorno o conjunto mayor en el que ese sistema opera. Es decir, todo lo que rodea y afecta al sistema. Seria el entorno = SUPRASISTEMA.

# PRINCIPALES NIVELES:
## Macrosistema
Es un sistema en si mismo, en que los elementos son a su vez **verdaderos sistemas**. Tambien recibe el nombre de metasistema, sistema exterior o sistema nivel de inclusion superior. Un ejemplo de macrosistema, es el universo, que engloba millones de galaxias.
## Sistema
Debe tener fronteras definidas y el objetivo.
## Subsistema
Es la parte inferior o parte de un sistema. Es conformado por los elementos que dentro de un elemento tiene un sistema tambien y su objetivos y fronteras pero se le denomina subsistema.
# Hasta que nivel se debe factorizar?
**La factorizacion o division en subconjuntos se efectua hasta el nivel que determina el objetivo del analisis final. Al llegar a ese punto estamos en Presencia del sistema elemental.**
# Sistema Elemental
Es el sistema de menor jerarquia y que no necesita ser factorizado para su analisis. Se define por sus entradas y salidas.
# Sistema de la caja negra
Es al nombre de denominacion que damos a un sistema al que no podemos factorizar, no lo conocemos internamente sino que conocemos su comportamiento exterior, que se visualiza por sus entradas y salidas.
- Una caja negra es un enfoque o modelo de analisis donde solo nos interesa el comportamiento del sistema (Entradas y Salidas), pero no nos preocupamos por como funciona internamente (los procesos y mecanismos internos).
